# 💰 MoneyMate – Smart Personal Expense Tracker

> **Manage your money smarter, track your expenses effortlessly, and build better financial habits.**

MoneyMate is a full-stack expense tracker web application built using the MERN stack. It enables users to securely manage their personal finances by tracking income, expenses, savings goals, and financial insights through a clean, responsive, and user-friendly interface.

Designed with simplicity, security, and usability in mind, MoneyMate helps users stay organized and make informed financial decisions.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/Krithika1031/MoneyMate.git
```

## Install Frontend

```bash
cd frontend
npm install
npm run dev
```

## Install Backend

```bash
cd backend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **backend** folder and add:

```env
PORT=5000

MONGO_URI=Your_MongoDB_Connection_String

JWT_SECRET=Your_JWT_Secret_Key

EMAIL=Your_Email_Address

EMAIL_PASSWORD=Your_Email_App_Password
```
